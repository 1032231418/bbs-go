## 模块
### server
> 为 https://mlog.club 提供数据支撑，基于`Golang`搭建。

### site
> https://mlog.club 的前端页面服务，基于`nuxt.js`搭建。


### admin
> https://mlog.club 的后台管理系统，基于`Vue` + `Element-UI`搭建。
